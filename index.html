jQuery Popup Overlay

jQuery plugin for responsive and accessible modal overlays and tooltips.

Try to change the width and height of browser window, or to rotate your tablet device, and also try to navigate with Tab key.

You can close the dialog by pressing the Esc key, or by clicking on the background outside the content area, or by clicking on the Close button.

If you don't want to show the outline around the container when focused, you can remove it with .overlay {outline: none}.

Features

Positioned with CSS only
Overlays are positioned with CSS, without any JavaScript offset calculations, enabling you to set flexible width and height to the overlay.
Suitable for responsive web design
Overlays will adapt the screen orientation and size if you set min-width and max-width correctly. It's fully customizable via CSS and media-queries.
Keyboard friendly
Modal overlays are keyboard accessible using Tab key.
Always on center, always visible
Position of the overlay is fixed at the center of the screen but if the height of the overlay exceeds the visible area – vertical scrolling will be automatically enabled to prevent the off-screen content from being unreachable.
Device agnostic
Works well on desktops, tablets, most modern phones and other devices.

Options

Name type default description
type overlay | tooltip 'overlay' popup type
action hover | click | focus | mouseover 'click' trigger event
horizontal center | left | right 'center' horizontal position
vertical center | top | bottom 'center' vertical position
background boolean true show of hide the background
color hex color '#000' background color
opacity float 0.4 background opacity
fade number 250 display the matched elements by fading them in and out (use `false` to disable)
keepfocus boolean true automatically focus Close Link when is opened, and focus Open Link when it's closed
escape boolean true close the modal with Esc key
blur boolean true closes the modal when clicked outside of it
autoopen boolean false open modal as soon as the plugin function is called
autozindex boolean false automatically set highest z-index to popup (this will parse the whole DOM first time it loads, so use this option only if you must – prefer CSS for setting z-index values)
onOpen function null callback function which will execute when the popup is opened
onClose function null callback function which will execute when the popup is closed
beforeopen function null callback function which will execute before the popup is opened
reposition boolean false popup reposition method – useful for repositioning overlays on callbacks after the popup contents change, to avoid content getting out of the screen

Browser Support

All modern browsers including IE7+.
